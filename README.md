# Krispy Kreme Account Generation With FREE Birthday Donut

## How It Works
- This script was made as a personal script when I was learning Python to create an account on Krispy Kreme
- The script will enter all the information through the valid xPaths and create the account towards my catchall
- This would also set my birthday as the next day, allowing me to take advantage of their free birthday donut offer

While the code is obselete now (as of 20th Feb 2026), this code was used by other companies who took this to a larger scale on other sites (where the code was not open sourced)
